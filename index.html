Formulario Geometrico 3D

Formule dei principali solidi geometrici

Cubo
Area LateraleAl = 4 * s2
Area TotaleAt = 6 * s2
VolumeV = s3
Diagonaled = s * √3
Spigolo (da Al)s = √(Al / 4)
Spigolo (da At)s = √(At / 6)
Spigolo (da V)s = ∛V
Spigolo (da d)s = d / √3
Parallelepipedo Rett.
Area di BaseAb = a * b
Area LateraleAl = 2p * c
Area TotaleAt = 2*Ab + Al
VolumeV = a * b * c
Diagonaled = √(a2 + b2 + c2)
Altezza (c)c = V / Ab oppure c = Al / 2p
Area di BaseAb = V / c
Perimetro di Base2p = Al / c
Prisma Regolare
Area LateraleAl = 2p * h
Area TotaleAt = 2*Ab + Al
VolumeV = Ab * h
Altezza (h)h = V / Ab oppure h = Al / 2p
Area di BaseAb = V / h
Perimetro di Base2p = Al / h
Cilindro
Area di BaseAb = π * r2
Area LateraleAl = 2 * π * r * h
Area TotaleAt = 2*Ab + Al
VolumeV = π * r2 * h
Altezza (h)h = V / (π * r2)
Raggio (r)r = √(V / (π * h))
Piramide Regolare
Area LateraleAl = (2p * a) / 2
Area TotaleAt = Ab + Al
VolumeV = (Ab * h) / 3
Apotema (Pitagora)a = √(h2 + r2)
Altezza (h)h = (3 * V) / Ab
Area di BaseAb = (3 * V) / h
Apotema (a)a = (2 * Al) / 2p
